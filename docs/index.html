R-JEPA

Reasoning Joint Embedding Predictive Architecture

A World Model for Text Reasoning inspired by Meta AI's V-JEPA. Predicts concepts, not tokens. Learns reasoning "physics" in latent space.

View on GitHub Quick Start

Core Features

Latent Space Prediction

Operates on LLM hidden states, not tokens. Learns stable relationships between reasoning concepts, like a blind person reading braille - pure conceptual understanding.

Multi-LLM Support

Works with Qwen3, Llama3, Mistral, DeepSeek, Phi families. Fast calibration adapts to new LLMs in 2-4 hours instead of days.

Validated Training Data

Trains only on verified correct reasoning chains. Mathematical validation (sympy), code execution tests, and teacher model verification ensure truth, not plausibility.

Continuous Learning

User feedback loop enables nightly retraining. The system improves with usage, like a living organism that learns from interactions.

Extended Benchmarks

Evaluated on GSM8K, MATH, HumanEval, MMLU (57 subjects), Big-Bench Hard (23 tasks), ARC, and HellaSwag.

Docker Ready

7 orchestrated services with Docker Compose. Production-ready deployment with health checks, logging, and monitoring.

Inference Modes

Production

RERANK

Generate K candidates, select the best by JEPA-loss. Combines logprob with coherence scoring for optimal reasoning chain selection.

Implemented

NUDGE

Guide generation with predicted latents using Logit Guidance. Token-by-token steering toward coherent reasoning paths.

Implemented

PLAN

Complete missing reasoning steps. Predict latents for gaps in the reasoning chain and decode them to text.

R-JEPA is inspired by V-JEPA (Meta AI, 2024) and A Path Towards Autonomous Machine Intelligence (Yann LeCun, 2022)
